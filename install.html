Harps - Installation

Harps isn't quite ready for prime-time yet, development of the generally available release is continuing. We are aiming for a Q1 2022 release.

For more information please refer re-read this site.

Linux

Once released Harps will be available through various package managers on Ubuntu, Alpine, Debian, etc. A Harps install includes the CLI and SDK.

Supported distributions:

Distribution Supported versions
Alpine 3.12, 3.13, 3.14
CentOS 7
Debian 9, 10, 11
Fedora 33, 34, 35
openSUSE 15
RHEL 7, 8
SLES 12 SP2, 15
Ubuntu 16.04, 18.04, 20.04, 21.04, 21.10

MacOS

Harps will be available through brew and other tools. We are targeting M1 support for Q2 2022.

Windows

No.
